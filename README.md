# Trionetix-Solutions

📌 Desafio proposto

O projeto consiste na criação de uma pequena rede corporativa com matriz e filial para atender às necessidades de TI de uma empresa de pequeno porte, incluindo servidor AD e banco de dados, priorizando desempenho, organização, segurança e escalabilidade.

📌 Conclusão do Projeto

Ao finalizar este projeto, pude comprovar na prática a importância de uma infraestrutura de rede bem planejada, organizada e segura. A criação da rede corporativa conectando a matriz e a filial, utilizando VLANs estruturadas, autenticação centralizada via Active Directory e comunicação estável através do túnel IPsec, me permitiu desenvolver uma solução completa e funcional, capaz de atender as necessidades reais de um ambiente corporativo.

Com a implementação dos servidores dedicados para serviços essenciais, como o AD e o banco de dados, consegui garantir maior desempenho, controle e segurança dentro da rede. A segmentação por VLANs tornou a gestão do tráfego mais eficiente, além de facilitar a administração e o monitoramento do ambiente.

O monitoramento do servidor de autenticação com Zabbix e Grafana também foi fundamental para acompanhar a saúde do serviço, permitindo identificar problemas de forma antecipada e garantindo maior confiabilidade ao sistema.

De forma geral, acredito que o projeto alcançou todos os objetivos estabelecidos. Consegui conectar as duas unidades de forma segura, organizar e segmentar a rede, centralizar a autenticação, permitir o acesso ao banco de dados entre matriz e filial e implementar um monitoramento eficiente. Esse trabalho me proporcionou um aprendizado significativo e reforçou minha capacidade de planejar, criar e administrar soluções de infraestrutura em ambientes reais.
