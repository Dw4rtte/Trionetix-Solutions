# Trionetix-Solutions

Ao concluir este projeto, percebo o quanto ele foi essencial para ampliar minha visão sobre infraestrutura de redes e segurança da informação. Desenvolver uma solução completa envolvendo matriz e filial, com Active Directory, banco de dados, FortiGate e monitoramento, me deu a oportunidade de colocar em prática conhecimentos que antes estavam apenas na teoria.

Cada parte — desde a criação das VLANs até a implementação do túnel IPsec e das políticas de firewall — exigiu atenção, organização e análise cuidadosa. Implementar o AD e integrar todo o ambiente, além de configurar o Zabbix e o Grafana para acompanhar o desempenho do servidor, me fez entender de forma muito mais profunda como ambientes corporativos funcionam no dia a dia.

Esse projeto não só reforçou minha capacidade técnica, como também fortaleceu minha confiança em atuar na área de TI. Foi um desafio importante e gratificante, que marcou meu desenvolvimento dentro do curso e mostrou que estou preparado para avançar para os próximos passos da minha carreira.

Mais do que uma entrega final, este trabalho representa o início de uma nova fase na minha trajetória profissional.
